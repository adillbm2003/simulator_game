Introduction

This guide explains how to use the Admin Dashboard for the Bermuda Home Energy Simulator. The dashboard gives you a real-time view of everyone who registers to use the simulator — including their name, email, institution, age group, gender, learner level, and area of interest.

You do not need any technical knowledge to use the dashboard. It works in any web browser on a computer, tablet, or phone.

✅
What you can do in the dashboard
  • See total number of registrations and simulations run
  • View every user's details in a searchable table
  • Watch new registrations appear live as people sign up
  • Export all registration data to a spreadsheet (CSV)
  • See which users ran the simulation, exported PDFs, or downloaded CSVs

1

2. Dashboard Overview

The Navigation Bar (top of screen)

At the top of the screen you will see a dark bar containing:

  • Bermuda Energy Simulator — Admin — the dashboard title
  • Green pulsing dot + "Live" — confirms you are receiving real-time updates. If it turns red and says "Reconnecting", your internet connection may have dropped briefly — it will reconnect automatically.
  • Your email address — confirms who is logged in
  • Sign Out button — click this when you have finished

The Stats Cards

Below the navigation bar are four summary cards showing the current totals:

CardWhat it shows
Total RegistrationsThe total number of people who have registered to use the simulator
Simulations RunHow many users actually ran the full 30-day simulation
PDFs ExportedHow many users downloaded their energy report as a PDF
CSVs DownloadedHow many users downloaded their simulation data as a spreadsheet
⚡
Real-time updates These numbers update automatically the moment a new user registers or runs the simulator. You do not need to refresh the page.

3. Viewing Registered Users

The Registered Users table shows every person who has filled in the registration form. The most recent registrations appear at the top. When a new user registers, their row will flash yellow briefly to draw your attention.

Searching for a specific user

1
Find the search box

Above the users table on the right, there is a box labelled "Search name or email…"

2
Type your search

Start typing a name or email address. The table filters instantly — no need to press Enter.

3
Clear the search

Delete what you typed in the search box and all registrations will reappear.

5. Exporting Data to a Spreadsheet

You can download all registration data as a CSV file which opens in Microsoft Excel, Google Sheets, or any spreadsheet application.

1
Find the Export button

In the Registered Users section, look for the "⬇ Export CSV" button on the right side of the section header.

2
Click Export CSV

A file named bermuda-registrations-YYYY-MM-DD.csv will automatically download to your computer.

3
Open in Excel or Google Sheets

Double-click the downloaded file to open it. Each row is one registered user with all their details in separate columns.

💡
Tip — Export regularly Export the CSV weekly or monthly as a backup. The database is secure and backed up automatically by Supabase, but having your own local copy is always good practice.

6. Signing Out

When you have finished using the dashboard, click the Sign Out button in the top-right corner of the navigation bar. This is especially important if you are using a shared computer.

⚠️
Shared computers Always sign out when using a shared or public computer. If you forget, anyone who opens the browser could access the registration data.

7. Frequently Asked Questions

The live dot is red and says "Reconnecting" — is something wrong?

This just means your internet connection briefly dropped. The dashboard will reconnect automatically within a few seconds. You do not need to refresh the page.

The table shows no data — is it broken?

If no one has registered yet, the table will be empty. It will populate as soon as the first user completes the registration form. If you believe data should be there, try signing out and signing back in.

Can two people be logged in at the same time?

Yes. Multiple staff members can have the dashboard open simultaneously and all will see the same live data.

The simulator.energy.bm address isn't working

The custom domain was configured on 14 June 2026 and may take up to 72 hours to become active worldwide. In the meantime, use simulatorgame.vercel.app — it is the same simulator.

A user says their registration didn't save

Ask them to check their internet connection and try again. If the problem persists, check the Supabase dashboard at supabase.com to confirm the project is active (it should show "ACTIVE HEALTHY" in green).

I forgot my admin password

Contact your IT administrator to reset your password in the Supabase Authentication panel.
